## 🎨 Design Standards

### Color Palette
```xml
Primary: #7B2CBF (purple_500)
Secondary: #E91E63 (pink_500)
Background: Gradient from #B3ABC3 to #FFFFFF
Text Primary: #212121
Text Secondary: #757575
Card Background: #FFFFFF
Success: #4CAF50
Error: #F44336
```

### Typography
- **Headers**: 24-32sp, Bold
- **Subheaders**: 18-20sp, Semi-bold
- **Body**: 14-16sp, Regular
- **Captions**: 12sp, Regular

### Spacing
- **Screen padding**: 24dp
- **Card margin**: 16dp
- **Element spacing**: 8-16dp

### Components
- **Buttons**: 56dp height, gradient background, white text
- **Cards**: 16dp corner radius, 2-4dp elevation
- **Input Fields**: OutlinedBox style, 12dp corner radius

## 📝 Implementation Notes

- All layouts use UTF-8 encoding without BOM
- Material Design 3 components throughout
- Responsive layouts for different screen sizes
- Accessibility labels and content descriptions
- RTL support maintained
